Now, every request that is made through $http to the URL /api/users.json will be stored in the default $http cache. The key for this request in the $http cache is the full-path URL. By passing the true parameter in the $http options, we’re telling the $http service to use the default cache. It’s useful to use the default cache if we don’t want to mess with the cache all that often. We can, however, manipulate the default $http cache if we need to (e.g., if another request we make without caching notifies us of a delta change, we can clear the request in the default $http request).
